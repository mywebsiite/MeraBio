# Mera Bio — Natural Beauty Website

A luxury Moroccan-inspired cosmetics website for **Mera Bio Natural Beauty**.

## 🌿 Features

- Responsive design (mobile, tablet, desktop)
- Interactive product tabs (Hair Oils, Body Scrubs, Bath Salts, Botanical Oils)
- Gift set gallery with hover overlays
- Product detail modals with WhatsApp ordering
- Contact form modal
- Newsletter signup
- Smooth scroll animations
- Back-to-top button
- Hamburger menu for mobile

## 🚀 Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your site will be live at `https://yourusername.github.io/merabio-website/`
